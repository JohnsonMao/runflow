# @bricks/core

Core interfaces and types for the Bricks workflow system.

## Installation

```bash
pnpm add @bricks/core
```

## Usage

```typescript
import type {
  IExpressionEvaluator,
  IExecutionContext,
  INodeExecutor,
  INodeExecutionContext,
} from "@bricks/core";
```

## Features

- Core interfaces for expression evaluation
- Execution context interfaces
- Node executor interfaces
- Type definitions
